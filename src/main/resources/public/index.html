Über den Kurs


Inhalte und Ablauf

In der Software Engineering Academy (SEA) lernen Sie Grundlagen der Java-Programmierung. In den ersten drei Monaten (Bootcamp) werden zunächst Grundlagen zu Linux, Bash und Git vermittelt, sowie eine ausführlichere Einführung in Java gegeben. Der Java-Teil beinhaltet folgende Themen:

  • Primitive Datentypen
  • Ablaufsteuerung
  • Klassen und Methoden
  • Arbeiten mit Maven
  • Vererbung
  • Interfaces
  • Generics
  • Arbeiten mit Datenbanken
  • Unit Tests
  • Lamda-Ausdrücke
  • Webserver mit Spring
  • Templating mit Thymeleaf
Nach dem Bootcamp können Sie Ihr erworbenes Wissen in einem echten Projekt testen und vertiefen.

Voraussetzungen

Grundsätzlich ist kein Vorwissen in Programmierung erforderlich, es hilft jedoch sehr. Wichtig ist, dass Sie Motivation und Gedult mitbringen.
